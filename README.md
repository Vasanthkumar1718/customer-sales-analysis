# Customer Sales Data Analysis

## Project Objective
Analyze customer sales data to identify trends, profitability patterns, and business insights.

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Analysis
- Sales trend over years
- Category-wise performance
- Region-wise profit analysis
- Discount vs profit relationship
- Product-level insights

## Key Insights
- Sales increased over time
- Technology category generates highest revenue
- Discount negatively impacts profit
- Some products generate high sales but low profit

## Business Recommendations
- Optimize discount strategies
- Focus on high-profit products
- Improve performance in low-profit regions

## Conclusion
This project shows how data analysis can help improve business decisions and profitability.
